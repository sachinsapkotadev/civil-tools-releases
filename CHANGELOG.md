# Changelog

All notable changes to Civil Tools will be documented in this file.

## [1.0] - 2025-06-13

### Initial Release

Full suite of civil engineering calculators:

- **Beam Analysis** — Shear force, bending moment, deflection for simply supported, cantilever, and continuous beams
- **Column Design** — Axial load capacity, slenderness checks, interaction diagrams for RC columns
- **Slab Design** — One-way and two-way slab design with moment and shear reinforcement
- **Foundation Design** — Isolated, combined, and strip footing design
- **Retaining Wall** — Stability analysis for cantilever and gravity retaining walls
- **Steel Design** — Section capacity checks, connections, and column buckling
- **Concrete Mix Design** — Mix proportioning per ACI and IS codes
- **Soil Mechanics** — Bearing capacity, settlement, and lateral earth pressure

### Features

- Offline-first architecture — no internet required
- Privacy-first — zero permissions
- Material Design 3 UI
- Dark mode support
